tPay 개발자 페이지

tPay 개발자를 위한 페이지 입니다.

tPay는 JTNet에서 운영하는 Payment Gateway 서비스로 tPay에서 가입하여 이용하실 수 있습니다. 해당 페이지는 tPay 를 이용하시는 개발자분들에게 정보를 제공하기 위한 페이지로, tPay 언제나 개발자분들을 위해 쉬운 개발 환경과 많은 정보를 제공하려고 노력하고 있습니다.

tPay는 오픈소스 개발자분들의 참여를 적극 지원합니다.

  • 저희 서비스를 보다 쉽게 사용하기 위해 개발된 오픈소스가 있으면 알려주세요.
  • tPay는 외부 개발자의 참여를 환영하고 지원합니다.

tPay SDK 및 메뉴얼

  • PHP SDK 공식 기술 지원 사이트를 통해 다운받으실 수 있습니다.
  • JSP SDK 공식 기술 지원 사이트를 통해 다운받으실 수 있습니다.
  • ASP SDK 공식 기술 지원 사이트를 통해 다운받으실 수 있습니다.
  • Ruby SDK 다운로드 / 개발 사이트에서 예제를 확인하실 수 있습니다.
  • Python SDK : 준비중입니다.
